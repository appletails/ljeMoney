# 文件夹介绍

```
只介绍src文件夹下内容，其他文件夹并没有做什么改变
```
- **/assets**

    ```
    静态资源文件夹，包括/js，/css，/img三大主要静态资源
    ```
    - js/api.js 各种请求都在这个文件里
    - js/filters.js 项目中所用到的过滤器都在这个文件里
    - css/global.scss 所有初始化的css样式，包括size的计算，配置颜色等
- **/components**
  
    ```
    用于盛放页面中所有用到组件的文件夹，其中 ItemPC.vue 是无用组件
    ```
